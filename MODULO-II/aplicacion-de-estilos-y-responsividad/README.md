# Aplicación de estilos y responsividad (Parte I)

Este material está pensado para **clases con estudiantes que recién comienzan**.

## Estructura

- `00-intro-css/` → Live Coding guiado: qué es CSS, selectores, cascada, caja, Flexbox y una media query.
- `01-aplicando-css-al-html/` → 4 ejercicios con HTML + CSS externo (buena práctica).

## Cómo usar

1. Descarga y descomprime la carpeta.
2. Abre el archivo `00-intro-css/index.html` en el navegador.
3. Luego abre `01-aplicando-css-al-html/index.html` para navegar los ejercicios.

## Sugerencia

- En cada ejercicio, abre:
  - `index.html` (estructura)
  - `assets/css/styles.css` (estilos)
- Explica siempre: **selector → propiedad → valor**.
